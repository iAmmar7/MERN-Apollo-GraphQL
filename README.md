# MERN-Apollo-GraphQL
CRUD app with MongoDB, Express, Node, Apollo Server, React, Apollo Client and GraphQL

## *Purpose*
:bangbang: *The purpose of this project is to integrate **GraphQL** with **Apollo** in a **MERN Stack** application* :bangbang:
